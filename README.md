# ARS Auto — сайт-витрина

Клиентский сайт для ars_autorentservice (Валенсия). MVP-версия: 6 машин-заглушек, русский язык, статический HTML без сборки.

## Структура

- `index.html` — главная: hero, процесс, сток, блок «что починили», FAQ
- `catalog.html` — каталог из 6 машин с фильтром по типу топлива
- `car.html` — страница конкретной машины (`?slug=...`) с таймлайном работ
- `about.html` — как мы работаем, 3 шага
- `contacts.html` — контакты, часы работы, карта Валенсии
- `assets/styles.css` — дизайн-система (светлая база, глубокий зелёный акцент, Fraunces + Inter)
- `assets/cars.js` — данные по 6 машинам
- `assets/app.js` — логика рендера (header, footer, каталог, страница машины)

## Фирменная фишка

На каждой странице машины — блок **«Что мы сделали в цехе»**: таймлайн работ с фото до / после, списком запчастей, именем мастера. Ни у одного конкурента (Clicars, OcasionPlus, AutoInSpain) такого нет.

## Деплой

Чистая статика. Vercel → подключить репозиторий → deploy. Никакой сборки не требуется.
