# ARS Auto — сайт-витрина

Клиентский сайт для ARS Auto (Валенсия). MVP-версия: 6 машин-заглушек, русский язык, статический HTML без сборки.

Позиционирование: продажа автомобилей с пробегом из Испании + услуга подбора и покупки на европейских аукционах под клиента.

## Структура

- `index.html` — главная: hero, процесс, сток, обещания, FAQ
- `catalog.html` — каталог из 6 машин с фильтром по типу топлива
- `car.html` — страница конкретной машины (`?slug=...`)
- `order.html` — форма подбора под клиента (генерирует Telegram-сообщение)
- `about.html` — как мы работаем, 3 шага
- `contacts.html` — контакты, часы работы, карта Валенсии
- `assets/styles.css` — дизайн-система (royal blue + dark navy, Inter)
- `assets/cars.js` — данные по 6 машинам
- `assets/app.js` — логика рендера (header, footer, каталог, страница машины)

## Деплой

Чистая статика. Vercel → подключить репозиторий → deploy. Никакой сборки не требуется.
